# Indicateur Accessibilité

Cette information n'est pas encore renseignée, elle correspondra à une donnée saisie à l'occasion d'audit d’accessibilité à venir ou déjà menés sur les modules applicatifs de type IHM seulement.

Les catégories seront les suivantes :

- **H :** pas de déclaration d’accessibilité
- **G :** déclaration + aucun audit enregistré
- **F :** déclaration + audit partiel avec résultat non conforme (\< 50% )
- **E** : déclaration + audit partiel avec résultat partiellement conforme
- **D** : déclaration + audit complet avec moins de 50% de conformité (non conforme)
- **C** : déclaration + audit complet avec une conformité en pourcentage comprise entre [50;75[ (partiellement conforme)
- **B :** déclaration + audit complet avec une conformité en pourcentage comprise entre [75;100[ (partiellement conforme)
- **A :** déclaration + audit complet conforme à 100% (conforme)
